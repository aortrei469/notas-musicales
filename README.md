# Notas Musicales

Juego de memoria musical donde el sistema propone secuencias de notas y el jugador debe repetirlas.

Copyright 2026 Arcadio Ortega

## Cómo jugar

1. Selecciona el número de jugadores (1-4)
2. Presiona **Iniciar**
3. Observa la secuencia de notas
4. Repite la secuencia en el mismo orden
5. ¡Gana puntos!

### Puntuación

- **1 jugador**: Cada acierto suma puntos. Cada fallo resta puntos.
- **2+ jugadores**: Al fallar, pasa el turno al siguiente jugador.

## Características

- 7 notas musicales con sonidos únicos
- Sistema de iluminación visual
- Modo para 1-4 jugadores
- Botones de pausa y reinicio
- Diseño responsivo para móvil

## Tecnologías

- HTML5
- CSS3
- JavaScript (Web Audio API)

## Ejecutar

Simplemente abre `index.html` en un navegador web.
